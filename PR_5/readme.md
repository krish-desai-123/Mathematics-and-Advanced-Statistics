## 🌟 Overview
Welcome to the **Calculative Foundation** project! This repository bridges the gap between raw mathematical theory and modern Machine Learning. It explores a simulated student performance dataset (across subjects like Math, Physics, Chemistry, and CS) using the core engines of Data Science: **Linear Algebra, Matrix Decompositions, and Dimensionality Reduction.**

Stop treating ML like a black box—this project unpacks the math that makes it all work! 💡

---

## 🛠️ Key Topics Explored

### 📐 Part A: Vector Fundamentals
* **Norms & Magnitude:** Calculating L1 (Manhattan) and L2 (Euclidean) norms.
* **Direction & Similarity:** Computing dot products and the exact angle between student performance vectors.
* **Orthogonality:** Cross products in 3D space and projecting one student's performance vector onto another.

### 🔲 Part B: Matrix Operations
* **Data as Matrices:** Transforming rows of student data into workable matrices.
* **Transformations:** Executing matrix addition, multiplication, transpositions, and calculating inverses and determinants.

### 🌐 Part C: Linear Transformations & Geometry
* Translating dry numbers into geometric space:
    * **1D:** Lines connecting data points.
    * **2D:** Planes grouping multiple points.
    * **N-1 D:** Hyperplanes acting as decision boundaries in higher-dimensional space.



### 🧩 Part D: Eigenvalues & Decompositions
* **Variance Under the Hood:** Extracting Eigenvalues and Eigenvectors from the covariance matrix to understand data spread.
* **Decompositions:** Breaking down complex matrices using **LU Decomposition** and **Singular Value Decomposition (SVD)**.

### 📉 Part E: Dimensionality Reduction
* **Principal Component Analysis (PCA):** Compressing 4-dimensional subject data down to 2 principal components while retaining key variance.
* **Linear Discriminant Analysis (LDA):** Classifying students into "Above Average" and "Below Average" categories by finding the optimal linear combination of features.



---

## 🧠 Why Does This Matter?
Linear algebra is the absolute bedrock of Artificial Intelligence, Computer Vision, and Predictive Modeling. By understanding how to manipulate vectors, decompose matrices, and reduce dimensions, you gain the foundational skills required to build, debug, and optimize advanced ML algorithms.
